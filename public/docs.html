Documentation

Welcome to the AniQuotes API documentation! This comprehensive guide will help you integrate beautiful anime quotes into your applications. From basic usage to advanced features, everything you need is here.

Introduction

AniQuotes API is a free, multilingual anime quotes service that provides instant access to thousands of curated quotes from your favorite anime series. With support for multiple languages and beautiful image generation, it's perfect for bots, websites, and mobile applications.

Key Features

  • 🔀 Random & Searchable Quotes: Get random quotes or search by anime/character
  • 🌍 Multilingual Support: English, Japanese, Hindi, and more languages
  • 🖼️ Image Generation: Create beautiful quote cards with anime styling
  • ⚡ Lightning Fast: Deployed on Vercel's edge network for global speed
  • 💎 No Authentication: Start using immediately without API keys
  • 📱 Developer Friendly: Clean REST API with comprehensive documentation
Pro Tip

The API is completely free and doesn't require authentication. You can start making requests immediately!

Authentication

AniQuotes API is completely free and open to use. No authentication, API keys, or registration is required. You can start making requests immediately to any endpoint.

No Authentication Required

Simply make HTTP requests to our endpoints. Perfect for prototyping, learning, and production use!

Rate Limiting

To ensure fair usage for all developers, we implement reasonable rate limits. Current limits are generous and suitable for most applications:

Endpoint Type Rate Limit Reset Period
Text Endpoints (V1, V2) 1000 requests Per hour
Image Generation 200 requests Per hour
Rate Limit Exceeded

If you exceed the rate limit, you'll receive a 429 Too Many Requests response. The limit resets every hour.

Quick Start

Get started with AniQuotes API in seconds. Here are some basic examples to help you begin:

1. Get a Random Quote

HTTP Request
GET https://aniquotesapi.vercel.app/v1/random
Response Example
{ "id": 42, "quote": "Believe in the me that believes in you!", "anime": "Gurren Lagann", "character": "Kamina", "language": "en", "credit": "GitHub/Shineii86" }

2. Search by Anime

HTTP Request
GET https://aniquotesapi.vercel.app/v1/anime?name=Naruto

3. Generate Quote Image

HTTP Request
GET https://aniquotesapi.vercel.app/v2/image?id=42&lang=en

Version 1 (V1) Endpoints

Version 1 endpoints provide basic quote retrieval functionality with simple, clean responses.

Random Quote

GET /v1/random
GET https://aniquotesapi.vercel.app/v1/random

Returns a random anime quote from our collection.

Quotes by Anime

GET /v1/anime
GET https://aniquotesapi.vercel.app/v1/anime?name=Naruto
Parameter Type Required Description
name string Required Name of the anime series

Quotes by Character

GET /v1/character
GET https://aniquotesapi.vercel.app/v1/character?name=Naruto
Parameter Type Required Description
name string Required Name of the character

Version 2 (V2) Endpoints

Version 2 introduces advanced features including multilingual support and powerful filtering options.

Multilingual Quotes

GET /v2/languages
GET https://aniquotesapi.vercel.app/v2/languages?lang=jp&anime=Naruto
Parameter Type Required Description
lang string Required Language code (en, jp, hi, etc.)
anime string Optional Filter by anime name
character string Optional Filter by character name

Image Generation

Generate beautiful, shareable quote images with anime-inspired designs. Perfect for social media, websites, and applications.

GET /v2/image
GET https://aniquotesapi.vercel.app/v2/image?id=42&lang=en
Parameter Type Required Description
id integer Required Quote ID to generate image for
lang string Optional Language for the quote text
Image Features

Generated images include beautiful backgrounds, proper typography, character names, anime titles, and watermarks. Images are returned as PNG format.

Error Handling

The API uses standard HTTP status codes to indicate success or failure. All error responses include a descriptive message and credit information.

Status Code Meaning Description
200 OK Request successful
400 Bad Request Invalid or missing parameters
404 Not Found No quotes found for the criteria
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error occurred

Error Response Format

Error Response Example
{ "error": "No quotes found for the specified anime", "credit": "GitHub/Shineii86" }
