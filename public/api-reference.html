API Reference

Complete reference for all AniQuotes API endpoints. Test each endpoint interactively with real-time responses and explore all available parameters.

Version 1 Endpoints

Get Random Quote

GET https://aniquotesapi.vercel.app/v1/random
GET

Returns a random anime quote from our curated collection. No parameters required - just call the endpoint and get a beautiful quote instantly.

Try it out
Click "Send Request" to see the response

Response Schema

Field Type Description
id integer Unique identifier for the quote
quote string The anime quote text
anime string Name of the anime series
character string Character who said the quote
language string Language code (e.g., "en", "jp")
credit string Attribution to the API creator

Get Quotes by Anime

GET https://aniquotesapi.vercel.app/v1/anime
GET

Get quotes from a specific anime series. Returns all available quotes for the specified anime.

Parameters

Parameter Type Required Description
name string Required Name of the anime series (e.g., "Naruto", "One Piece")
Try it out
Enter an anime name and click "Send Request"

Get Quotes by Character

GET https://aniquotesapi.vercel.app/v1/character
GET

Get all quotes said by a specific character across different anime series.

Parameters

Parameter Type Required Description
name string Required Name of the character (e.g., "Naruto", "Luffy")
Try it out
Enter a character name and click "Send Request"

Version 2 Endpoints

Get Multilingual Quotes

GET https://aniquotesapi.vercel.app/v2/languages
GET

Get quotes in specific languages with optional filtering by anime or character. Supports multiple languages including English, Japanese, Hindi, and more.

Parameters

Parameter Type Required Description
lang string Required Language code (en, jp, hi, etc.)
anime string Optional Filter by anime name
character string Optional Filter by character name
Try it out
Select a language and click "Send Request"

Generate Quote Image

GET https://aniquotesapi.vercel.app/v2/image
GET

Generate beautiful, shareable images of quotes with anime-inspired designs. Perfect for social media, websites, and applications.

Parameters

Parameter Type Required Description
id integer Required Quote ID to generate image for
lang string Optional Language for the quote text
Try it out
Enter a quote ID and click "Generate Image"
