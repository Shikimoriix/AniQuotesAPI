Get perfectly curated random anime quotes or search by specific anime series and characters. Our intelligent system ensures variety and quality.

Quick Start

Get up and running with AniQuotes API in seconds. No authentication required!

Get a random quote 
curl https://aniquotesapi.vercel.app/v1/random
Get quotes by anime 
curl "https://aniquotesapi.vercel.app/v1/anime?name=Naruto"
Generate quote image 
curl "https://aniquotesapi.vercel.app/v2/image?id=42&lang=en"
