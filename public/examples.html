JavaScript Examples

Get Random Quote (Fetch API)

Simple example using modern JavaScript fetch API to get a random anime quote.

random-quote.js
// Get a random anime quote async function getRandomQuote() { try { const response = await fetch('https://aniquotesapi.vercel.app/v1/random'); const quote = await response.json(); console.log(`"${quote.quote}"`); console.log(`- ${quote.character} (${quote.anime})`); return quote; } catch (error) { console.error('Error fetching quote:', error); } } // Usage getRandomQuote();

Python Examples

Simple Quote Fetcher

Basic Python script using requests library to get anime quotes.

quote_fetcher.py
import requests import json class AnimeQuoteFetcher: BASE_URL = "https://aniquotesapi.vercel.app" def get_random_quote(self): """Get a random anime quote""" try: response = requests.get(f"{self.BASE_URL}/v1/random") response.raise_for_status() return response.json() except requests.exceptions.RequestException as e: print(f"Error fetching quote: {e}") return None def get_quotes_by_anime(self, anime_name): """Get quotes from specific anime""" try: params = {"name": anime_name} response = requests.get(f"{self.BASE_URL}/v1/anime", params=params) response.raise_for_status() return response.json() except requests.exceptions.RequestException as e: print(f"Error fetching quotes: {e}") return None # Usage fetcher = AnimeQuoteFetcher() # Get random quote quote = fetcher.get_random_quote() if quote: print(f'"{quote["quote"]}"') print(f"- {quote['character']} ({quote['anime']})") # Get quotes from Naruto naruto_quotes = fetcher.get_quotes_by_anime("Naruto") if naruto_quotes: print(f"Found {len(naruto_quotes)} Naruto quotes!")
