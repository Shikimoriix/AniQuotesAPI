Adding New Quotes

Follow these simple steps to add new anime quotes to our collection:

1

Fork the Repository

Fork the AniQuotes API repository on GitHub to create your own copy where you can make changes.

2

Locate the Data File

Navigate to the data/quotes.json file for English quotes, or the appropriate language file in data/languages/.

3

Add Your Quote

Add your quote following the exact JSON structure shown below:

Quote JSON Structure
{ "id": 9999, "quote": "Your awesome anime quote here", "anime": "Anime Title", "character": "Character Name", "language": "en" }
4

Submit Pull Request

Create a pull request with a clear description of the quotes you've added. Include the anime series and number of quotes added.

Development Setup

Ready to contribute code? Here's how to set up the development environment:

Setup Commands
# Clone your fork git clone https://github.com/YOUR_USERNAME/AniQuotesAPI.git cd AniQuotesAPI # Install dependencies npm install # Start development server npm run start # Run tests npm test
1

Create a Feature Branch

Always create a new branch for your feature or bug fix: git checkout -b feature/your-feature-name

2

Make Your Changes

Write clean, well-documented code. Follow the existing code style and add tests for new functionality.

3

Test Your Changes

Run npm test to ensure all tests pass and your changes don't break existing functionality.

4

Submit Pull Request

Push your changes and create a pull request with a detailed description of what you've implemented.

Community Guidelines

We strive to maintain a welcoming and inclusive community. Please follow these guidelines:

Quality Standards

Ensure quotes are accurate, properly attributed, and from legitimate anime sources. Avoid duplicates and maintain high quality standards.

Be Respectful

Treat all community members with respect. Be constructive in feedback and discussions. We're all here to improve the API together.

Content Guidelines

Keep content appropriate and family-friendly. Avoid quotes with explicit content or offensive language.
